Variables

Variables, එහෙමත් නැත්නම් විචල්‍ය කියන්නෙ computer එකේ මතකය තුල නම් කරන ලද තැනක්. සරලව කිව්වොත් variable එකක් කියන්නෙ නමක් සහ අගයක් ඇති දෙයක්. අපිට පුළුවන් යම් අගයකට නමක් දීලා ඒක variable එකක් විදිහට program එකේ වෙනත් තැනකදි පාවිච්චි කරන්න. මේක හරියට අපි ගණිතයේදි 3.141592654 කියන මතක තියා ගන්න සහ ලියන්න අමාරු, ඒ වගේම නිතර පාවිච්චි වෙන, සංඛ්‍යාවට “π” (pi) කියලා පාවිච්චි කරනවා වගේ.

අපි python වලදි variable එකකට අගයක් ලබාදෙන්න පාවිච්චි කරන්නෙ = ලකුණ. මේ උදාහරණයේදි මම pi කියන variable එකට 3.14592654 කියන අගය ලබා දෙනවා.

pi = 3.14592654

අපිට මේ අගය වෙනස් කරන්න ඕනෙ වුනොත් ආයෙත් මේ ක්‍රමයම පාවිච්චි කරන්න පුළුවන්. මේ වතාවෙදි මම pi කියන variable එකට 3 කියන අගය දෙනවා.

pi = 3

variable එකකට නමක් දෙනකොට අපි කැමති නමක්, වචනයක් පාවිච්චි කරන්න පුළුවන්. කොහොමවුණත් මේ නම variable එකේ අගය පැහැදිලි කරන නමක් නම් වඩා හොඳයි. මොකද x වගේ නමක් අද අපිට මතක තිබුණට තව මාස ගාණකින් අපේ code එක කියවද්දි අපිට x කියන්නෙ මොකද්ද කියල අමතක වෙලා තියෙන්න පුළුවන්. ඒත් pi වගේ නමක් අපිට මතක තියාගන්නට ඕනෙත් නෑ.

python වලදි variable එකක නම case-sensitive, ඒ කියන්නෙ නමේ අකුරු CAPITAL, simple වීම මත එක නමක් තවත් නමකින් වෙනස් වෙනවා. උදාහරණයක් විදිහට, pi, Pi, PI, pI කියන්නෙ එකිනෙකට වෙනස් නම් තුනක්. ඒ නිසා අපිට pi කියලා variable එකක් හදලා PI විදිහට ඒක පාවිච්චි කරන්න බෑ. සම්ප්‍රදායක් විදිහට python වල variables වල නම් ලියන්නෙ simple අකුරින් (මේක වෙනස් වෙන තැන් තියෙනවා). variable එකක නමක ඉලක්කම් තියෙන්න පුළුවන්, හැබැයි ඉලක්කමකින් නමක් පටන් ගන්න බෑ. (ඉංග්‍රීසි) අකුරු, ඉලක්කම් වලට අමතරව, underscore (_) ලකුණත් පාවිච්චි කරන්න පුළුවන්. ඒත්, සෘණ ලකුණ (-), ධන ලකුණ (+), තරුව (*) වගේ characters පාවිච්චි කරන්න බෑ.

වලංගු variable නම් වලට උදාහරණ,

language = "Python"
language2 = "Java"
easy_language = "Python"
hardLanguage = "Java"

Strings

Strings කියන්නෙ වචන, වාක්‍යය වගේ දත්ත වලට. මේක දත්ත වර්ගයක්. කලින් උදාහරණෙදි language, language2, easy_language, hardLanguage වගේ variable අයත් වෙන්නෙ Strings වර්ගයට. String වර්ගයට අයත් දත්තයක් (String එකක්) දක්වන්න Python වල විශේෂ ක්‍රමයක් පාවිච්චි කරනවා, මොකද නැත්නම් මේ දත්තය string එකක් බව computer එකට අඳුරගන්න ක්‍රමයක් නැති නිසා. string එකක් අපි උඩු කොමා (") දෙකක් මැද්දෙ දක්වනවා. කලින් උදාහරණෙ එන, "Python", "Java" කියන්නෙ strings. තනි උඩු කොමාවක් වෙනුවට අපිට යුගල උඩු කොමා (") වුණත් පාවිච්චි කරන්න පුළුවන්. උදා: "Python", "Java". මේ ක්‍රම දෙකේ කිසිම වෙනසක් නෑ. දෙපැත්තෙ තියෙන උඩු කොමා දෙක string එකට අයිති වෙන්නෙ නෑ. ඒවා පාවිච්චි වෙන්නෙ string එකේ ආරම්භය හෝ අවසානය සලකුණු කරන්න.

උඩුකොමා භාවිතය

උඩු කොමා පාවිච්චියේදි මතක තියා ගන්න ඕනෙ දේ තමයි පටන් ගන්න කොමාවෙන්ම string එක ඉවර කරන්නත් ඕනෙ බව. ඒ කියන්නෙ තනි උඩු කොමාවකින් string එකක් පටන් ගත්තොත් එතන ඉඳන් මුණ ගැහෙන පළවෙනි තනි උඩු කොමාව වෙනකම් සැලකෙන්නෙ string එකක් විදිහට. යුගල උඩු කොමා වලටත් මේ කාරණාව අදාලයි.

අපිට string එකක තනි උඩු කොමාවක් (") ඇතුලත් කරන්න ඕනෙ නම් අපි string එක දක්වන්න ඕනෙ යුගල උඩුකොමා දෙකක් මැද්දේ,

string_one = "It"s Python."

string එකක යුගල උඩු කොමා (") ඇතුලත් කරන්න ඕනෙ නම් ඒ string එක දක්වන්න ඕනෙ තනි උඩු කොමා දෙකක් මැද්දෙ,

string_two = "He said, "It is Python"."

යම් උඩු කොමාවකට කලින් backslash (\) ලකුණක් එකතු කලොත් ඒ කොමාව string එකක ආරම්භයක් හෝ අවසානයක් හෝ දක්වන්න පාවිච්චි වෙන්නෙ නෑ. ඒ වගේ වෙලාවට python interpreter එක ඒක සලකන්නෙ තවත් එක character එකක් විදිහට.
උදාහරණයක් විදිහට අපිට කලින් උදාහරණ දෙක මේ විදිහටත් ලියන්න පුළුවන්,

string_one = "It\"s Python"
string_two = "He said, \"It\"s Python\"."

මේ උදාහරණයේ තියෙන strings දෙක python interpreter එක තේරුම් ගන්නේ කලින් උදාහරණ දෙක තේරුම් ගත්තු විදිහටමයි.

Indexing

string එකක තියෙන හැම character එකකටම අදාල index එකක් තියෙනවා. ඒ කියන්නෙ අංකයක්. හරියටම කිව්වොත් string එකක හැම character එකක්ම අංකනය කරලා තියෙනවා. මේ අංකනය කිරීම පටන් ගන්නෙ 0න්. උදාහරණයක් විදිහට "Python" කියන string එක ගත්තොත් මේ string එකේ හැම අකුරක්ම තියෙන්නෙ එක්තරා අංකයක් ඇති කොටුවක් ඇතුලෙ. මේ අංකය හඳුන්වන්නෙ “index” එක කියලා.

        +---+---+---+---+---+---+
String: | P | y | t | h | o | n |

Index :   0   1   2   3   4   5

Built-in functions

Python වලදි function එකක් කියන්නෙ නැවත නැවත පාවිච්චි කරන්න පුළුවන් codes කොටසක් වගේ දෙයක්. අපි ගණිතයෙදි ශ්‍රිත පාවිච්චි කරන විදිහටම තමයි programming වලදිත් ශ්‍රිත (functions) පාවිච්චි කරන්නෙ. උදාහරනයක් විදිහට f(x) = x + 3 කියල ශ්‍රිතයක x වෙනුවට 5 යෙදුවම (input කළාම) අපිට 8 කියන උත්තරේ ලැබෙනවා (output එක). මේ විදිහටම python වල function එකකටත් අපිට input එකක් (මේ inputs වලට කියන්නෙ arguments කියලා) දීල output එකක් අරගන්න පුළුවන්. හරියට program එකක් ඇතුලෙ තියෙන තවත් පොඩි පොඩි programs වගේ. python program එකක නිතරම කෙරෙන දේවල් කරන්න functions (python හදන අය) වෙනම ලියලා තියෙනවා.

print()

print() function එක python වල තියෙන නිතරම පාවිච්චි වෙන function එකක්. print() function එක කරන්නෙ ඒකට දෙන inputs (arguments) screen එකේ display කරන එක. අපිට argument එකක් විදිහට කෙලින්ම අගයක් දෙන්න පුළුවන්, එහෙමත් නැත්නම් variable එකක් දෙන්නත් පුළුවන්.

greet = "Hello"
print(greet)
print("World!")

output එක:

Hello
World!

len()

len() function එකෙන් කරන්නෙ යම් කිසි container (උදා: string, array, etc…​) එකක තියෙන items ගණන ගණන් කරලා දෙන එක. string එකක් `len()`function එකේ argument එක විදිහට දුන්නොත් අපිට ලැබෙන්නෙ ඒ string එකේ තියෙන characters ගණන.

මේ උදාහරණෙදි මම lang කියලා variable එකකට "Python" කියන අගය දෙනවා. ඊට පස්සෙ lang_length කියලා තවත් variable එකක අගය විදිහට lang string එකේ තියෙන characters ගණන දෙනවා. අන්තිමට අපි lang_length එක print කරලා බලමු.

lang = "Python"
lang_legth = len(lang)
print(lang_length)

String methods

Python කියන්නෙ object oriented programming language එකක්. ඒ වගේම python වල තියෙන හැමදෙයක්ම මොකක් හරි object එකක්. ඒ වගේම හැම object එකකටම type (data type; දත්ත වර්ගය) තියෙනවා. මේ වෙද්දි අපි string data type එක අඳුර ගෙන තියෙනවා.

අපි කලින් උදාහරණ වලදි පාවිච්චි කරපු "Python" කියන object එක අයිති වෙන්නෙ str කියන වර්ගයට (str කියන්නෙ කෙටියෙන් string කියන එක). තවත් විදිහකට කිව්වොත්,"Python" කියන්නෙ string object එකක්. අපි මේ object එක variable එකකට දුන්නම ඒ variable එකත් string වර්ගයට අයත් වෙනවා.

අපි කලින් කතා කළා functions ගැන. අපි එතැනදි කතා කලේ ඕනෙම object එකකට පාවිච්චි කරන්න පුළුවන් function දෙකක් ගැන. Python වල එක එක Object වර්ගයට විශේෂ වුණු functions තියෙනව. මේවාට කියන්නෙ “Methods” කියලා. Methods ක්‍රියාකරන්නෙ මොකක් හරි object එකක් මත (මේ කොටසෙදි කතා කරන්නෙ str object එක මත ක්‍රියාකරන methods ගැන).

යම් කිසි object එකක් මත method එකක් ක්‍රියාකරවන්නෙ, object එකට පස්සෙ තිතකුත් (.) ඊට පස්සෙ method එකේ නමත් ඊට පස්සෙ වරහන් දෙකකුත් යොදලා. method එකේ arguments තියෙනවා නම් ඒවා වරහන් අතරෙ කොමා වලින් වෙන් කරලා දක්වන්න පුළුවන්. ඊලඟ උදාහරණ වලදි මේ කිව්වෙ මොකද්ද කියලා පැහැදිලි වෙයි.

lower() method

lower() method එකෙන් කරන්නෙ යම් කිසි string object එකක copy එකක් simple අකුරු වලට හරවලා output එක විදිහට return කරන එක.

lang = "Python"

output:

python

Strings එකට ඈඳීම (string concatenation)

strings දෙකක් එකට ඈඳන්න (එකතු කරන්න, බද්ධ කරන්න) අපිට එකතු කිරීමේ ලකුණ (+) පාවිච්චි කරන්න පුළුවන්. හරියට ඉලක්කම් දෙකක් එකතු කරනවා වගේ.

str()

ඊලඟ කොටස් වලදි අපි ඉගෙන ගන්නවා සංඛ්‍යාත්මක දත්ත ගැන (numbers). numbers සහ strings කියන්නෙ එකිනෙකට වෙනස් දත්ත වර්ග දෙකක්. ඒ නිසා අපි string එකකට number එකක් එකතු කරනවා නම්, අපිට් මුලින් ඒ number එක string එකක් බවට හරවන්න වෙනවා. මේ වැඩේට අපිට str() කියන function එක පාවිච්චි කරන්න පුළුවන්. str() function එක කරන්නෙ string නොවෙන වස්තුවක් string එකක් බවට හැරවීම. උදා:

version = 3
print("This is Python " + str(version) + ".")

output:

This is Python 3.

version කියන variable එක string එකක් බවට හරවන්නෙ නැතුව මේක කරන්න ගියොත්,

version = 3
print("This is Python " + version + ".")

output:

Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: Can't convert 'int' object to str implicitly

මේ වගේ error එකක් ලැබෙනවා.

format() Method

format() method එක string එකක් සකස් කරද්දි ගොඩක් වටින method එකක් කියලා කියන්න පුළුවන්. අපිට පුළුවන් string එකක සඟල වරහන් ({}) පාවිච්චි කරලා යම් යම් තැන්(format fields) සලකුණු කරලා, ඒ ඒ තැන් වලට format() method එකට දෙන arguments ඇතුළු කරන්න.

සාමාන්‍යයෙන් මේ ඇතුළු කිරීම සිද්ධ වෙන්නෙ පළවෙනි සඟල වරහන් යුගල තියෙන තැනට පලවෙනි argument එකත් දෙවැනි තැනට දෙවැනි argument එකත් විදිහට.

print("I {} Python.".format("Love"))
print("{} {} {}".format("I", "Love", "Python"))

output:

I love Python.
I love Python.
අපි method එකකට හරි function එකකට හරි arguments එකකට වඩා වැඩි ගණනක් දෙනවා නම් ඒවා කොමා වලින් වෙන් කරනවා, format("I", "Love", "Python")

අපිට අවශ්‍ය නම් යම් format field එකකට යෙදෙන්න ඕනෙ argument එක දක්වන්නත් පුළුවන්. කරන්න තියෙන්නෙ සඟල වරහන් අතරෙ කීවෙනි argument එකද කියලා දක්වන එක. argument ගණන් කිරීමත් පටන් ගන්නෙ 0න්. ඒ නිසා {0} කියන්නෙ පළවෙනි argument එක, {1} කියන්නෙ දෙවැනි argument එක…​

print("I {0} {1}, {1} {0}s me.".format("Love", "Python"))

output:

I Love Python, Python Loves me.

කලින් වගේම අපිට මේකට variables පාවිච්චි කරන්නත් පුළුවන්.

first = "I"
second = "Love"
third = "Python"
sentence = "{} {} {}.".format(first, second, third)
print(sentence)

output:

I Love Python.

ඒ වගේම format() method එක පාවිච්චි කරනවා නම් අපිට str() function එක අවශ්‍ය වෙන්නෙත් නෑ. කලින් උදාහරණෙ මේ විදිහට ලියන්නත් පුළුවන්,

version = 3
print("This is Python {}.".format(version))

output:

This is Python 3.

format specifications

අපිට අවශ්‍ය නම් මේ string format එකට තවත් දේවල් එකතු කරන්න පුළුවන්. ඒවට කියන්නෙ format specifications කියලා. format specification එකක් පටන් ගන්නෙ දෙතිතකින් (:). උදාහරණයක් විදිහට හිතන්න අපිට string එකක් සඟල වරහන් තුලට යොදන කොට අවම වශයෙන් characters N ගණනක පළලක් තියෙන විදිහට format කරන්න ඕනෙ වුණා කියලා. අපිට පුළුවන් සඟල වරහන් ඇතුලෙ : ට පස්සෙ අවශ්‍ය N සටහන් කරලා මේක දක්වන්න. මේත් එක්ක කලින් වගේම අපිට argument එකේ අංකයත් දක්වන්න පුළුවන්. උදාහරණයක් විදිහට {0:8} කියන්නෙ “format() හි පළවෙනි argument එක අරගෙන අඩුතරමේ characters 8ක පළලක් වත් එන විදිහට මෙතැන යොදන්න” කියන එකයි. {1:8} කියන්නෙ “format() හි දෙවැනි argument එක අරගෙන අඩුතරමේ characters 8ක පළලක් වත් එන විදිහට මෙතැන යොදන්න” කියන එකයි. උදාහරණයක් විදිහට, අපිට වගුවක් print කරන්න ඕනෙ නම් මේ විදිහට format specifications පාවිච්චි කරන්න පුළුවන්.

print("|{0:8} | {1:8}|".format("Fruit", "Quantity"))
print("|{0:8} | {1:8}|".format("Apple", 3))
print("|{0:8} | {1:8}|".format("Orange", 10))

output:

|Fruit    | Quantity|
|Apple    |        3|
|Orange   |       10|

අපිට අවශ්‍ය නම් මේ strings වල alignment වුණත් අපිට අවශ්‍ය විදිහට හදන්න පුළුවන්. left align කරන්න < ලකුණත්, center align කරන්න ^ ලකුණත්, right align කරන්න > ලකුණත් පාවිච්චි කරන්න පුළුවන්. කිසිම ලකුණක් පාවිච්චි නොකලොත් right align විදිහට තමයි print වෙන්නෙ. උදා:

print("|{0:8} | {1:<8}|".format("Fruit", "Quantity"))
print("|{0:8} | {1:^8}|".format("Apple", 3))
print("|{0:8} | {1:>8}|".format("Orange", 10))

output:

|Fruit    | Quantity|
|Apple    |    3    |
|Orange   |       10|

format field එකක අපිට අවශ්‍ය නම් data type එකක් දක්වන්නත් පුළුවන්. මේකෙ සුලභම භාවිතය තමයි දශම සංඛ්‍යාවක දශමස්ථාන සංඛ්‍යාව දැක්වීම. .Nf හරහා අපිට සංඛ්‍යාවක් දශමස්ථාන N ගණනකට දැක්වෙන විදිහට හදන්න පුළුවන්. උදා:

print("|{0:^8}|{1:^8}|".format("x", "f(x)"))
print("|{0:^8.3f}|{1:^8.3f}|".format(1.02334, 10.33425))
print("|{0:^8.3f}|{1:^8.3f}|".format(2.33455, 15.99355))
print("|{0:^8.3f}|{1:^8.3f}|".format(3.56985, 19.22598))
print("|{0:^8.3f}|{1:^8.3f}|".format(4, 22))

output:

|   x    |  f(x)  |
| 1.023  | 10.334 |
| 2.335  | 15.994 |
| 3.570  | 19.226 |
| 4.000  | 22.000 |

User inputs ලබා ගැනීම

හුඟක් අවස්ථා වල අපි ලියන programs වලට user ගෙන් inputs අරගන්න වෙනවා. standard input හරහා inputs අරගන්නවා නම් අපිට input() function එක පාවිච්චි කරන්න පුළුවන්. standard input (stdin) එක කියන්නෙ හුඟක් වෙලාවට command prompt එක හරහා keyboard එකෙන් type කරන inputs. input() function එක කරන්නෙ ඒකට ලබා දෙන argument එක display කරලා, ඊට පස්සෙ user දෙන input එක return කිරීම. උදා:

name = input("Enter your name: ")
print("Hello, {}.".format(name))

output:

Enter your name: Python
Hello, Python.

input() function එකෙන් අපිට ලැබෙන්නෙ str output එකක්.
