Python ගැන

මොකද්ද මේ python කියන්නේ

මේ කොටසෙ කතා කරන දේවල් තේරෙන්නෙ නැතිනම් ඒක ප්‍රශ්නයක් කරගන්න එපා. මේ ටික මගහැරලා ඉස්සරහට යන්න.

Python කියන්නේ පරිවර්තිත (interpreted) ක්‍රමලේඛන භාෂාවක්. ක්‍රමලේඛන භාෂාවක් පාවිච්චි කරලා තමයි අපි පරිගණකයට යම් කාර්යයක් කරන්න උපදෙස් දෙන්නෙ. හැබැයි පරිගණකයට අපි මේ ලියන දේවල් කෙලින්ම කියවන්න බෑ. කොටින්ම කියනවා නම් සිය ගාණක ක්‍රමලේඛන භාෂා තිබුනත් පරිගණකයට ඒ එකක් වත් තේරුම් ගන්න බෑ. ඒ නිසා අපි program එකක් ලියනවා නම්, අපිට කැමති භාෂාවකින් ඒක ලියලා, යම් පරිවර්තනයක් කරන්න වෙනවා, ඒ කියන්නෙ compile කරන්න වෙනවා. compile කරද්දි වෙන්නෙ අපි ඉංග්‍රීසි අකුරින් ලියපු codes පරිගණකට තේරුම් ගන්න පුළුවන් එකම භාෂාව වෙන (binary codes කියලා හඳුන්වන) 0 සහ 1 ඒවා ගොඩකට (වෙනම file එකකට මේවා ලියැවෙනවා) පරිවර්තනය වෙන එක. මේ 0 සහ 1 ඒවා ලියැවෙන file එක තමයි Windows වලදි නම් .exe1 file එකක් කියලා හඳුන්වන්නෙ.

මෙහෙම compile කරපු program එකක තියෙන ප්‍රශ්නෙ තමයි ඒක compile කලේ Windows වලදි නම් ඒක run2 කරනවා නම් Windows වලම run කරන්න වීම. ඒ කියන්නේ ඒ program එක Linux වලදි run කරන්න බෑ (එහෙම ඕනෙ නම් Linux වලට ගැලපෙන විදිහට compile කරන්න ඕනෙ.)

Python කියන්නේ මෙහෙම compile කිරීමක් (සම්පූර්ණ compile කිරීමක්) අවශ්‍ය වෙන්නෙ නැති භාෂාවක්. python වලදි compile කිරීමක් වෙනවා, හැබැයි ඒ compile කිරීම කරන්නෙ compiler එකක් නෙමෙයි, interpreter එකක්. Interpreter එක කරන්නෙ program එකේ එක් එක් විධානය binary code එකට පරිවර්තනය කිරීම. මේ වැඩේ සිද්ධ වෙන්නෙ program එක run වෙන මොහොතේදීම නිසා වෙනම compile කිරීමක් පෙනෙන්නෙ නෑ. program එක run වෙන මොහොතක් පාසා මේ පරිවර්තනය සිද්ධ වෙන නිසා සමහර විට මේක කාලෙ නාස්තියක් වගේ පේන්න පුළුවන්. ඒත් මේකෙ තියෙන වාසිය තමයි Windows වල වගේම linux වලත්, Mac වලත් එකම code එක කිසිම වෙනසක් නැතුව run කරන්න පුළුවන් වීම.

Python interpreter එක install කරගැනීම ගැන විස්තර ඊලඟ පරිච්ඡේදයෙ සඳහන් වෙනවා.

1. exe කියන්නෙ executable කියන එක. ↩
2. “run” කරනවා කියන එකෙන් මම අදහස් කරන්නෙ program එක ක්‍රියාත්මක කරනවා කියන එක. “execute” කරනවා කියලා කියන්නෙත් මේකට ↩
