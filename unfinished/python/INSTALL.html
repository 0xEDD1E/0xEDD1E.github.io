Python install කිරීම

Python install කිරීම

Python වලින් programming කරන්න කලින් අපිට python interpreter එක install කරගන්න ඕනෙ. ඒ වගේම අපිට codes ලියන්න text editor එකකුත් ඕනෙ කරනවා. text editor එකක් විදිහට අපිට python වලටම හදපු IDLE editor එක පාවිච්චි කරන්න පුළුවන්. Windows වලදි නම් notepad එකට වඩා IDLE පාවිච්චි කරන එක හොඳයි. Notepad වලින් ලිව්වොත් පොඩි පොඩි ප්‍රශ්න ඇති වෙන්න ඉඩ තියෙනවා.

Windows වලට python install කිරීම

Windows වලදි සාමාන්‍ය program එකක් install කරන පිලිවෙලටම python interpreter එක install කරගන්න පුළුවන්. මුලින්ම තමන්ට ගැලපෙන installer (පහල බලන්න) එක python site එකෙන් download කරගන්න ඕනෙ. මේ පිටුවෙ download කරගන්න installers කිහිපයක්ම තියෙනවා. ඒ අතරින් x86 කියන වචනෙ තියෙන්නේ 32-bit OS වලට ගැලපෙන installers වල, x86-64 කියලා තියෙන්නේ 64-bit OS වලට ගැලපෙන installers වලට. web-based installer කියලා තියෙන installer එකකින් install කරන්න නම් internet connection එකක් ඕනෙ කරනවා. executable installer එක තමයි සාමාන්‍ය installer එක (මේක download කරනවා නම් හොඳයි) මේ installer එක download කරගත්තට පස්සෙ සාමාන්‍ය විදිහටම install කරන්න පුළුවන්.

Linux සඳහා

හුඟක් linux මෙහෙයුම් පද්ධති වල python කලින්ම install කරලා තියෙනවා. ඒ නිසා සමහර විට අලුතින් python install කරන්න වුවමනාවක් වෙන්නෙ නෑ (අලුත් Debian, Ubuntu, Fedora වගේ OS එකකනම් python3 කලින්ම install කරල තියෙනවා). දැනටමත් python install කරලා තියෙනවද කියලා බලාගන්න පුළුවන් මේ command එක terminal එකේ දීලා,

$ python3

මේ command එකෙන් මේ වගේ ප්‍රතිඵලයක් ලැබෙනවා නම්, දැනටමත් python install කරලා තියෙන්නෙ.

Python 3.5.2 (default, Nov 17 2016, 17:05:23) 
[GCC 5.4.0 20160609] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>>

මේ විදිහෙ output එකක් නැතුව (මෙතැන තියෙන ඉලක්කම් වෙනස් වුණාට ප්‍රශ්නයක් නෑ) command not found වගේ output එකක් ලැබෙනවා නම් python install කරලා නෑ. එහෙම වෙලාවට python install කරන්න හොඳම ක්‍රමය තමයි package manager එක හරහා install කරගන්න එක.

එහෙම නැත්නම් Python source එක download කරගෙන ඒක compile කරගන්නත් පුළුවන්. මුලින්ම python site එකෙන් Gzipped source tarball හරි, XZ compressed source tarball කියන දෙකෙන් එකක් download කරගන්න. මම XZ compressed source එක download කරගන්නවා. ඊට පස්සෙ terminal එකෙන් download කරපු tar file එක extract කරගන්න.

$ tar xvf ~/Downloads/Python-3.6.0.tar.xz
