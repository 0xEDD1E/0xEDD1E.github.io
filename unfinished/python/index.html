හැඳින්වීම

මේ පොත ලියැවෙන්නේ python ගැන කිසිම දෙයක් දන්නෙ නැති, කලින් කවදාවත් programming කරලා නැති අය වෙනුවෙන්. ඒ කියන්නේ programming වලට සම්පූර්ණයෙන්ම අලුත් අයට. මේ පොතේ කරුණු පුළුවන් තරම් සරලව පැහැදිලි කරන්න මම උත්සහ කරන්නම්. කොහොමවුනත් programming ඉගෙන ගන්න තියෙන්නේ එක ක්‍රමයයි, ඒ තමයි අනිත් අයගෙ codes කියැවීමයි, තමන් තනියෙන් codes ලිවීමයි. ඒ නිසා මේක පොත කියෙව්ව පමණින් ඔයාට programming පුළුවන් වෙන එකක් නෑ, ඔයා තනියෙන් programs ලියන්නත් ඕනෙ. ඒ නිසා මේ පොත කියවන අතරෙ මේකෙ එන උදාහරණ ඔයා උත්සහ කරලා බලන්න ඕනෙ. හැබැයි මේකෙ තියෙන codes copy/paste කරන්න එපා. ඒ ක්‍රමේ හරියන්නේ නෑ.

මේ පොතේ උදාහරණ වලට මම පාවිච්චි කරන්නේ Python 3 සංස්කරණය. ඒ නිසා මේ උදාහරණ සමහර විට Python 2 වලදි වැඩ නොකරන්න පුළුවන්. හරියටම කිව්වොත් මම පාවිච්චි කරන්නේ Python 3.5.2 සංස්කරණය.

පොතේ පාවිච්චි වෙන රටා

මේ පොත ලියද්දි මම පාවිච්චි කරපු වාක්‍ය රටා කිහිපයක් තියෙනවා. ඒ කියන්නේ codes, outputs, inputs වගේ දේවල් පොතේ අනිත් කොටස් වලින් වෙන් කරලා අඳුරගන්න ලේසි වෙන ක්‍රම කිහිපයක්.

මේ පොතේ මේ විදිහට,

# prints a greeting
print("Hello, World!")

යම් කොටසක් ලියලා තියෙනවා නම්, ඒ ලියලා තියෙන්නේ python codes. මේ කොටසක අන්තර්ගතය සමහර විට එක එක පාට වලින් දැක්වෙන්න පුළුවන්. එහෙම පාට පාටින් දැක්වෙන්නේ කියවන්න ලේසි වෙන්න. හැබැයි මේවා ඔයා text editor එකක (notepad, gedit) type කරන කොට මේ වගේ පාට නොවෙන්න පුළුවන්. ඒක ප්‍රශ්නයක් නෙමෙයි වැදගත් වෙන්නේ code එක හරියට ලිවීමයි, පාට වැදගත් වෙන්නෙ නෑ.

computer එකේ output එකක් ලියන කොට මම මෙන්න මේ වගේ ක්‍රමයක් පාවිච්චි කරනවා.

Hello, World!

කලින් කොටසෙ වගේමයි හැබැයි මෙතැනදි කිසිම පාට කිරීමක් වෙන්නේ නෑ.

ඒ වගේම සමහර විට අපි ලියන program එකකට input එකක් දෙන්න අපිට සිද්ධ වෙන්න පුළුවන් ඒ වගේ වෙලාවට මම මෙන්න මේ වගේ ක්‍රමයක් පාවිච්චි කරනවා,

Input your name: User
Welcome, User

මෙතැනදි තද කළු අකුරින් ලියල තියෙන්නේ අපි type කරපු දේවල්. අනිත් දේවල් computer එකේ outputs.

ඒ වගේම මේ පොත ලියද්දි terminal (command prompt, cmd) එකේ commands දෙන අවස්ථා මට පැහැදිලි කරන්න වෙනවා. ඒ වගේ අවස්ථා වල මම මෙන්න මේ වගේ ක්‍රමයක් පාවිච්චි කරනවා.

$ cd Python-3.6.0/
$ ./configure --prefix=$HOME/py3
... දිග output එකක් ...

මෙතැන $ ලකුණින් දැක්වෙන්නේ prompt එක. prompt එකක් තියෙන පේලියෙ තියෙන දේවල් විතරයි ඕගොල්ලො terminal (cmd) එකේ දෙන්න ඕනෙ. prompt එකක් නැති පේළි වල තියෙන්නේ command එකේ outputs එක. මෙතැන මම තවත් දෙයක් දක්වල තියෙනවා. උඩ තියෙන කොටසෙ ඇල අකුරින් තියෙන්නේ( ... දිග output එකක් ... කියන කොටස) මම ලියන දේවල්, ඒවා කිසිම command එකක, program එකක output එකක්, input එකක් නෙමෙයි. සමහර දේවල් පැහැදිලි කරන්න මම එකතු කරපු වාක්‍ය තමයි ඒ ඇල අකුරින් තියෙන්නේ.

පොත කියවන්න ඕනෙ කරන පෙර දැනුම

මම මුලින්ම කිව්වා වගේ මේ පොත ලියන්න programming වලට සම්පූර්ණයෙන්ම අලුත් අයට, programming පටන් ගන්න තැනක් විදිහට. ඒ නිසා මේ පොතේදි මම programming ගැන සංකීර්ණ දේවල් කතා කරන්න යන්නෙ නෑ. ඒක සමහර විට කියවන කෙනාට පහසුවක් වෙන්නත් පුළුවන්, සමහර විට අපහසුවක් වෙන්නත් පුළුවන්. උදාහරණයක් විදිහට මම ඊලඟ පරිච්ඡේදෙදි python ගැන කතා කරද්දි මේ විදිහට සඳහන් කරනවා,

Python කියන්නේ පරිවර්තිත (interpreted) ක්‍රමලේඛන භාෂාවක්. ක්‍රමලේඛන භාෂාවක් පාවිච්චි කරලා තමයි අපි පරිගණකයට යම් කාර්යයක් කරන්න උපදෙස් දෙන්නෙ ....

මේතැනදි මම ක්‍රමලේඛන භාෂාව කියන දේ පැහැදිලි කලත් පරිවර්තිත කියන වචනෙ, "පරිවතිත කියන්නේ..." කියල පැහැදිලි කරන්න යන්නේ නෑ, මොකද එහෙම කරන්න ගියොත් මට ප්‍රධාන කාරණාවෙන් බැහැරව ගොඩක් දුර යන්න වෙන නිසා. ඒ වෙනුවට සමහර විට ඒ ගැන දල අදහසක් ඊලඟ ඡේද වලදි කියන්න ඉඩ තියෙනවා, එහෙමත් නැත්නම් ඒ ගැන පැහැදිලි නොකර ඉස්සරහට යන්නත් පුළුවන් (පැහැදිලි නොකළා කියලා ඒකෙන් python පිළිබඳ දැනුමට හානියක් වෙන එකක් නෑ).
